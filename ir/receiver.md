## リモコンの仕様
- 回路図

```
受信機
                / / 赤外線
               V V                                        Monitor
          +---------------+  Vout  +----------------+    +--------------------------------------------------------------------+
VCC o-----| 赤外線モジュール |--------| pin8           |    | Code = 555AF148688B.                                               |
GND o-----|               |        |                | => | Bit data[49] = 0101010101011010111100010100100001101000100010115   |
          +---------------+        | arduino uno v3 |    |   :                                                                |
                                   |                |    |   :                                                                |
                                   +----------------+    +--------------------------------------------------------------------+
```

- 型番

|パラメータ|値|
|:-|:-|
|赤外線受信モジュール|型番不明（中国製）|
|電圧範囲|2.4 - 5.0[V]|
|周波数|38[kHz]|

|パラメータ|値|
|:-|:-|
|赤外線LED|TLN115A|


- 受信モジュールのピンアサイン

|足番||
|:-|:-|
|1|Vout|
|2|GND|
|3|VCC|

```
+-----+
| up  |
+-----+
  \_/
|  |  |
|  |  |
1  2  3
```
